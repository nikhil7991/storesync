## Installation

Install storesync with pnpm (recommended)

```bash
  pnpm i
  pnpm run dev
```

Install storesync with npm

```bash
  npm i
  npm run dev
```

## Tech Stack

```bash
React
Vite - Bundler
Redux - State management

Tailwind - CSS framework
Shadcn ui - UI Library
```
